Frequently Asked Questions

Do I need a build step to use this library?
No. Every component runs directly in the browser via a CDN-loaded Tailwind stylesheet and vanilla JS files.
Is this library accessible?
Yes. All interactive components implement the W3C ARIA Authoring Practices patterns and pass keyboard navigation and screen reader testing.
Can I use this in a React or Vue project?
The HTML structure and CSS patterns are framework-agnostic. The JS would need porting, but the component architecture translates directly.
